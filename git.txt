폴더에서 우클릭으로 폴더경로 지정된 상태의 git 실행 가능.

ls -al
	해당경로의 폴더, 파일 열람

git init
	해당 경로에 git 설정. (.git)이라는 폴더가 생성됨.
	
cd -폴더명
	해당 폴더명으로 경로 설정. 'cd -.git'을 입력하면 (.git)폴더 안으로 경로설정됨.

git add 파일명
    해당 파일을 git에 추가함.

git config user.name "사용자명"
    사용자명을 추가한다.

git config user.email "이메일주소"
    사용자의 이메일주소를 추가한다.

git commit -m "코멘트"
    커밋메시지를 포함하여 커밋 실시.
    사용자 이름, 이메일주소, git add   3가지가 모두 되어야 한다.

git status
    현재 git의 상태확인.

git add .
    수정된 모든 파일을 add한다.
    
mkdir 폴더명
    폴더명의 이름을 가진 폴더를 만든다.

touch 파일명
    (추정)파일을 만드는 듯.
